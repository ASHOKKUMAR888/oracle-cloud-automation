# Oracle Database 23ai Automation POC

## Overview

This project demonstrates a complete **Proof of Concept (POC)** for deploying **Oracle Database 23ai Free** in a Docker container on **AWS EC2 (Amazon Linux 2023)**. The project covers database deployment, administration, SQL automation, Python integration using the Oracle Database Python driver, CRUD operations, stored procedures, and source code management with Git and GitHub.

The objective is to provide a lightweight, reproducible environment for Oracle database development and automation using container technology.

# Deployment Steps

## 1. Launch AWS EC2

* Create an EC2 instance running Amazon Linux 2023.
* Configure the security group:

  * SSH (22)
  * Oracle Database (1521) if remote access is required.

---

## 2. Install Docker

```bash
sudo dnf update -y
sudo dnf install docker -y
sudo systemctl enable docker
sudo systemctl start docker
sudo usermod -aG docker ec2-user
```

---

## 3. Pull Oracle Database Image

```bash
docker pull container-registry.oracle.com/database/free:latest
```

---

## 4. Run Oracle Database

```bash
docker run -d \
--name oracle23ai \
-p 1521:1521 \
-p 5500:5500 \
-e ORACLE_PWD=YourPassword \
container-registry.oracle.com/database/free:latest
```

---

## 5. Verify Database

```bash
docker ps
```

```bash
docker logs oracle23ai
```

# Database Setup

Connect as SYSDBA and create the application schema.

Example:

```sql
CREATE USER CLOUDADMIN IDENTIFIED BY password;

GRANT CONNECT, RESOURCE TO CLOUDADMIN;

ALTER USER CLOUDADMIN QUOTA UNLIMITED ON USERS;
```

# Create Sample Table

```sql
CREATE TABLE CUSTOMERS (
    CUSTOMER_ID NUMBER GENERATED BY DEFAULT AS IDENTITY,
    CUSTOMER_NAME VARCHAR2(100),
    EMAIL VARCHAR2(100),
    CITY VARCHAR2(50),
    PRIMARY KEY (CUSTOMER_ID)
);
```

# Stored Procedures

Example procedure:

```sql
CREATE OR REPLACE PROCEDURE GET_CUSTOMER_COUNT AS
v_total NUMBER;
BEGIN
    SELECT COUNT(*) INTO v_total FROM CUSTOMERS;
    DBMS_OUTPUT.PUT_LINE('Customers: ' || v_total);
END;
/
```
